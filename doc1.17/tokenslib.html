Using TokensLib¶

In this chapter we will learn how to use the TokensLib library.

Introduction¶

TokensLib is a simple library written in Ring.

The library provide classes to get Ring tokens from Ring source code.

RingTokens Class¶

The class comes with the next methods:

fromFile(cFileName)
fromString(cString)
getTokens() --> aTokensList
printTokens()

Example¶

load "tokenslib.ring"

func main
        oTokens = new RingTokens {
                fromFile("hello.ring")
                PrintTokens()
                ? Copy("=",50)
                fromString("? 1+1")
                PrintTokens()
        }

Output:

Keyword     : SEE
Literal     : Hello, World!
EndLine
==================================================
Operator    : ? (23)
Number      : 1
Operator    : + (1)
Number      : 1
EndLine
